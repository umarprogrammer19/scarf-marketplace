# Sapphire-Inspired Premium Fashion Layout

## Layout Structure

### Top Bar (AnnouncementBar)
- Maroon background (#8B1538)
- Phone icon + number on left
- Centered announcement text: "FREE DELIVERY ON ORDERS ABOVE PKR 3,000"
- Clean, minimal design

### Header (Navbar)
- Logo on left (ELEGANCE text logo)
- Two-row category navigation in center:
  - Row 1: New Arrivals, Winter Collection, Silk Collection, Velvet Collection
  - Row 2: Karandi Collection, Khaddar Collection, Summer Collection, Printed Lawn Collection
- Right-aligned icons: Search, User, Wishlist (badge: 3), Cart (badge: 2)
- Maroon accent color (#8B1538) on hover

### Category Controls
- Clean white background
- Grid toggle icons centered (2, 3, 4 columns)
- Sort dropdown right-aligned
- Minimal spacing, professional look

### Product Grid
- Max width: 1400px, centered
- Default: 4 columns on desktop
- Responsive: 3 columns tablet, 2 columns mobile
- Large gap between products (gap-8)
- Premium whitespace

### Product Cards
- 3:4 aspect ratio images
- Discount badge: top-right, maroon background
- Sold out overlay: grey with centered badge
- Hover effects: scale + shadow
- Clean typography, light font weights
- Price display with strikethrough for original price
- "New Arrival" label for new products

## Design Principles
- Luxurious and clean aesthetic
- Large whitespace for premium feel
- Subtle hover animations
- Maroon (#8B1538) as primary accent
- Light, elegant typography
- Minimal borders and shadows
- Focus on product imagery

## Responsive Breakpoints
- Mobile: 2 columns (default)
- Tablet (md): 3 columns
- Desktop (lg): 4 columns

## Color Palette
- Primary Accent: #8B1538 (Maroon)
- Background: White (#FFFFFF)
- Text: Gray-900, Gray-700
- Badges: Maroon for discount, Gray for sold out
